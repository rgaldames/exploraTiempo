# Explora Tiempo 🌎🕰️

**Explora Tiempo** es una aplicación educativa para Android diseñada para ayudar a los niños a comprender los conceptos del tiempo y las estaciones del año, integrando elementos de la historia de Chile y herramientas interactivas.

**Explora Tiempo** no es solo un calendario, es una herramienta de identidad. La aplicación integra hitos históricos clave para fomentar el sentido de pertenencia en los niños:

-   **Chile Continental**: Hitos como las Fiestas Patrias (18 de sept) y Glorias Navales (21 de mayo).
-   **Chile Insular (Rapa Nui)**: Conmemoración del "Acuerdo de Voluntades" (incorporación de la isla).
-   **Archivo Juan Fernández**: Datos sobre el descubrimiento y la historia del archipiélago.

> **Nota pedagógica**: El sistema TTS utiliza una entonación amigable para explicar estos sucesos históricos, adaptando el lenguaje para que sea comprensible para menores de 6 a 12 años.



## 🌟 Características Principales

-   **Aprendizaje del Tiempo**: Secciones interactivas para aprender los días de la semana, los meses, las estaciones y la estructura de un año.
-   **Interfaz 3D Interactiva**: Botones con efectos visuales avanzados, incluyendo:
    -   Efecto de "hundimiento" al presionar (3D).
    -   Barrido de color horizontal animado de izquierda a derecha.
-   **Audio Educativo (TTS)**: Integración de **Text-to-Speech** para anunciar nombres, descripciones y datos curiosos de forma auditiva.
-   **Historia de Chile**: Contenido educativo que incluye fechas históricas relevantes tanto para Chile continental como insular (Rapa Nui y Juan Fernández).
-   **Estadísticas del Año**: Visualización del "Mapa de las 52 Semanas" y datos sobre años bisiestos.
-   **Monetización**: Integración de AdMob mediante banners para soporte del proyecto.

## 🛠️ Tecnologías Utilizadas

-   **Lenguaje**: [Kotlin](https://kotlinlang.org/)
-   **UI Framework**: [Jetpack Compose](https://developer.android.com/jetpack/compose) (Arquitectura moderna y declarativa)
-   **Diseño**: *Material Design 3* con componentes personalizados.
-   **Servicios**: 
    -   Google Play Services Ads (AdMob).
    -   Android TextToSpeech API.
-   **Herramientas**: Android Studio.

## 🚀 Instalación y Uso

1.  Clona este repositorio:
    ```bash
    git clone https://github.com/tu-usuario/explora-tiempo.git
    ```
2.  Abre el proyecto en **Android Studio**.
3.  Asegúrate de tener instalado el SDK de Android (API 34 o superior recomendado).
4.  Sincroniza el proyecto con Gradle.
5.  Ejecuta la aplicación en un emulador o dispositivo físico.
